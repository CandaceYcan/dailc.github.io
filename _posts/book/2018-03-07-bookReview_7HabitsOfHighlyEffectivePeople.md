## 简介

七个习惯：

```js
1. Be Proactive

2. Begin With The End In Mind

3. Put First Things Fist

4. Think Win/Win

5. Seek First To Understand, Then To Be Understood

6. Synergize

7. Sharpen The Saw
```

个人领域的三个习惯：

- Be Proactive，确保自己的主动性，主动积极的人才是对自己生命负责的人

- Begin With The End In Mind，其实就是确立目标，当你假想你的生命即将结束时，你才会知道自己真正想要做什么

- Put First Things Fist，重要的事情优先，分清事情的轻重缓急

公共领域的三个习惯：

- Think Win/Win，考虑双赢，利人利己才是最优做法

- Seek First To Understand, Then To Be Understood，换位思考，先理解别人，再想着被人理解

- Synergize，统合综效，避免分割，避免单打独斗，很多事情综合起来考虑才会有最优结果

自我完善：

- Sharpen The Saw，不断更新，做一个终身的学习者

## 一些想法

可以说，是`17`年读的最有意义的一本书

这本书买下来时选了英文版版，硬着头皮读下去的，每天花点时间，磨磨蹭蹭大半年才完整读完

这本书的名字是：`The Seven Habits of Highly Effective People -- Restoring the Character Ethic`，
感觉看起来像是传统的成功学，鸡汤学，
但是，当你开始读后，会发现，这本书更多的是指导你如何重新塑造自己，
就像其中提及的`Restoring the Character Ethic`

看书至少有两个阶段：

- 知道

- 理解

像这本书，如果仅仅是粗略的过一遍，可能就是知道而已，知道每一个习惯，
知道它的详细描述，甚至能想象到它到来的改变，
但是，如果没有真正的实践，是永远也不会理解的

而我，在读这本书时，恰好自己也正在坚持着某些习惯，
于是，在某种程度上有一定的重合，共鸣，
我也尝试着用里面的几条准则，确实是深有体会

当然，现阶段主要还是在个人领域以及自我完善方面，
公共领域上的实践还有所欠缺

我深刻的记得，里面有这样一段话：

```js
请想想，
是否有一件事情，
是你做了后会给自己带来很大益处的，
如果有，
那么现在就开始做
```

于是，我找到了这样一件事情，并且付诸了实践，
从结果上，也确实给我带来了很大的收益，
至少改变了我接下来数年内的生活，
甚至会影响整个职业规划，人生规划

其它的，
譬如重要事情优先准则，
分清事情的轻重缓急，将不重要的事情砍掉，
譬如主动性，
要做一个积极主动的人，
而不是消极被动的人，
譬如不断更新，
生命不息，学习不止，
等等

总的来说，
如果发现和这本书里面的某些准则有共鸣，
那么付诸于行动就是，
肯定会有收获的

后续准备再读几遍，随着人生阅历的增加，肯定也还会有新的感悟
